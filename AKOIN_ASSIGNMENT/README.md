# Institute of Digital Risk (IDR) - Website & Brand Design

## 📋 Project Overview

This project contains a complete brand identity and responsive website for the Institute of Digital Risk (IDR), an industry-led training and deployment institute focused on digital, cyber, and AI risk management.

## 🎨 Deliverables

### 1. Logo Design
- **logo-icon.svg** - Icon-only variant (perfect for favicons and mobile headers)
- **logo-full.svg** - Icon + text "Institute of Digital Risk" variant
- **LOGO_DESIGN_NOTES.md** - Detailed explanation of design choices

### 2. Website
- **index.html** - Semantic HTML5 structure
- **styles.css** - Responsive CSS with modern design (no frameworks)
- **script.js** - Vanilla JavaScript for interactivity

## 🚀 Features

### Technical Implementation
✅ Semantic HTML5 (header, nav, main, section, footer)
✅ Responsive layout using CSS Flexbox and Grid
✅ Sticky navigation with smooth scrolling
✅ Mobile-responsive design (desktop + mobile)
✅ Accessible color contrast
✅ Google Fonts (Inter)
✅ Hover states for buttons and navigation
✅ No CSS frameworks (pure vanilla code)

### Design Features
- Modern, minimalist aesthetic
- Orange (#FF6B35), white, and black color palette
- Smooth animations and transitions
- Parallax effects
- Scroll-reveal animations
- Interactive form with validation
- Mobile hamburger menu

## 📱 Sections

1. **Hero Section** - Mission statement with CTA buttons
2. **About IDR** - Institute overview with statistics
3. **Service Pillars** - Academy, Innovation & Incubation, Advisory Services
4. **Pipeline Visualization** - Train → Hire → Innovate → Deploy
5. **Community** - Target audience segments
6. **Contact Form** - Registration of interest form

## 🎯 How to Use

### Option 1: Open Directly
Simply open `index.html` in any modern web browser (Chrome, Firefox, Safari, Edge)

### Option 2: Local Server (Recommended)
```bash
# Using Python 3
python -m http.server 8000

# Using Node.js
npx http-server

# Then open: http://localhost:8000
```

## ✨ Key Design Principles

1. **Structure** - Geometric cube represents organized risk management
2. **Resilience** - Solid foundation with interconnected elements
3. **Innovation** - Modern gradient and dynamic animations
4. **Professionalism** - Clean typography and balanced layout
5. **Accessibility** - High contrast ratios and semantic HTML

## 📝 Form Functionality

The contact form includes:
- Name field (required)
- Email field (required, validated)
- Interest dropdown (Training/Hiring/Advisory/Partnership)
- Message textarea (required)
- Submit button with success feedback

Currently configured for demonstration. To connect to a backend:
1. Update the form action in `index.html`
2. Modify the submit handler in `script.js`
3. Add server-side validation
