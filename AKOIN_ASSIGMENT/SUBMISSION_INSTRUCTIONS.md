# 📤 ИНСТРУКЦИЯ ПО ОТПРАВКЕ ЗАДАНИЯ

## 🚀 ОТПРАВКА ЧЕРЕЗ GITHUB

### ШАГ 1: Настройка GitHub Pages

1. Перейдите на: https://github.com/Arthurs-hub/Internships-Assignments
2. Нажмите **Settings** (вверху справа)
3. В левом меню выберите **Pages**
4. В разделе **Source**:
   - Branch: выберите `main`
   - Folder: выберите `/ (root)`
5. Нажмите **Save**

После этого сайт будет доступен по адресу:
```
https://arthurs-hub.github.io/Internships-Assignments/AKOIN_ASSIGMENT/
```

⏱️ Подождите 2-3 минуты, пока GitHub Pages развернет сайт.

### ШАГ 2: Проверка работы сайта

Откройте в браузере:
```

2. Перейдите к заданию
3. Нажмите "Submit Assignment"
4. В поле для ссылки вставьте:
   ```
