# Quick Start Guide - IDR Project

## 🚀 Getting Started

### View the Website
1. Open `index.html` in any web browser
2. The website is fully responsive - try resizing your browser!
3. Test on mobile by opening in your phone's browser

### Export Logo to PNG

**Option 1 - Simple Method (Recommended):**
1. Open `logo-export-simple.html` in your browser
2. Right-click on any logo image
3. Select "Save Image As..." and save the PNG file
4. Or click the "Download PNG" button

**Option 2 - Direct SVG Use:**
- Use `logo-icon.svg` and `logo-full.svg` directly
- SVG files work in all modern applications
- They scale perfectly to any size without quality loss
