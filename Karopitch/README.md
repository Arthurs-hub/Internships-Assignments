# Karo Pitch - Landing Page

## 🚀 Project Overview

This is a modern, responsive landing page designed for **Karo Pitch** - a platform connecting early-stage founders from across India with investors through curated pitch events.

## 📋 Assignment Details

**Organization:** KaroStartup  
**Project:** Demo Website for Karo Pitch  
**Platform:** Custom HTML/CSS/JavaScript (No AI builder used - built from scratch for maximum customization)

## 🎨 Design Approach

### Why This Design?

1. **Modern & Clean UI**: Used contemporary design trends with gradient accents, smooth animations, and card-based layouts
2. **Startup Ecosystem Vibe**: Incorporated vibrant colors (purple/indigo gradient) that resonate with the tech startup community
3. **Investor-Focused**: Clear value proposition, professional presentation, and structured information flow
4. **Mobile-First**: Fully responsive design that works seamlessly on all devices
5. **User Experience**: Intuitive navigation, clear CTAs, and smooth scrolling for better engagement

### Design Features

- **Hero Section**: Bold headline with gradient text, dual CTAs, and trust indicators (stats)
- **About Section**: Clear explanation of the platform's mission with visual hierarchy
- **How It Works**: 4-step process with numbered cards for easy understanding
- **Category Cards**: Visual representation of eligible startup types
- **Featured Startups**: Showcase section with dummy startup profiles
- **Investor Section**: Highlights different types of investors on the platform
- **Social Proof**: KaroStartup credentials and community stats
- **Strong CTAs**: Multiple conversion points throughout the page

### Color Palette

- Primary: `#6366f1` (Indigo)
- Secondary: `#8b5cf6` (Purple)
- Accent: `#ec4899` (Pink)
- Dark: `#0f172a`
- Background: `#f8fafc`

### Typography

- Font Family: Inter (Google Fonts)
- Modern, clean, and highly readable

## 🛠️ Technology Stack

- **HTML5**: Semantic markup
- **CSS3**: Modern styling with flexbox, grid, animations
- **JavaScript**: Smooth scrolling and interactive elements
- **No frameworks**: Pure vanilla code for fast loading
